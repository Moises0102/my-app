La página web

En este caso la pagina web ha sido codificada con un lenguaje de marcas, o dicho de otra forma con HTML5
Sin embargo usar unicamente HTML5 hubiese sido muy sencillo, por lo que la página esta hecha principalmente con:

ANGULAR

Angular es un framework que usa typeScript para realizar unas páginas web dinámicas y más vistosas
La principal diferencia respecto a HTML5 es que angular no necesita recargar la pestaña para cambiar de sección. Haciendo que sea visiblemente más rápida para el usuario.


A su vez la págia web hace uso de herramientas más comunes como SCSS, un lenguaje de hoja de estilos en cascada, que nos ha ayudado a darle formato y diseño a los elementos de la web

The Fallen

El juego ha sido desarrollado con Unreal Engine 4, un motor de video gratuito desarrollado por la desarrolladora de videojuegos Epic Games.
Es totalmente gratiuto, al menos lo esencial, fue lanzado en 1998, aunque la versión que nosotros usamos fue lanzada el 5 de marzo de 2015, de forma gratuita.

Este motor usa un lenguaje propio

"The Blueprint Visual Scripting system in Unreal Engine is a complete gameplay scripting system based on the concept of using a node-based interface to create gameplay elements from within Unreal Editor."

Dicho por los propios desarrolladores, es una codificacion basada en nodos, los cuales hacen el desarrollo mucho más dinámico y sencillo a la hora de empezar en este lenguaje
Este es un ejemplo de uno de los blueprints que hemos desarrollado.
